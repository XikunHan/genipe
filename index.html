Introduction¶

The gwip (Genome-Wide Imputation Pipeline) module provides an easy an efficient way of performing genome-wide imputation analysis using the three commonly used softwares PLINK, SHAPEIT and IMPUTE2. It also provides a useful standalone tool to perform statistical analysis on imputed (dosage) data (such as linear, logistic or survival regressions, or SKAT analysis of rare variants).
